# Daycare Management System

A premium, modern dashboard for managing students, staff, and automated reminders.

## 🚀 Quick Start

To run the application locally, follow these steps:

### 1. Start the Backend
Open a terminal and run:
```bash
npm run dev --prefix backend
```
*The API will be available at: http://localhost:5001*

### 2. Start the Frontend
Open a **new** terminal and run:
```bash
npm run dev --prefix frontend
```
*The dashboard will be available at: http://localhost:5173 (standard Vite port)*

## 🛠 Tech Stack
- **Frontend**: React (Vite), Vanilla CSS (Premium Glassmorphism)
- **Backend**: Node.js, Express
- **Database**: SQLite (Stored in `/database/daycare.sqlite`)
